# 动作示意图上传目录

将找到的图片放入此目录，**文件名必须与下表一致**（推荐 PNG，也支持 JPG/WebP，需同步改 `config.js` 里的 `image` 扩展名）。

| 文件名 | 动作名称 |
|--------|----------|
| `neck_chin_tuck.png` | 下巴回收训练 |
| `neck_side_bend.png` | 颈部侧屈拉伸 |
| `chin_tuck.png` | 收下巴训练 |
| `scapular_retraction.png` | 肩胛后缩训练 |
| `thoracic_extension.png` | 胸椎伸展训练 |
| `cat_cow.png` | 猫牛式脊柱松动 |
| `pelvic_tilt.png` | 骨盆后倾训练 |
| `bird_dog.png` | 鸟狗式核心训练 |
| `dead_bug.png` | 死虫式核心训练 |
| `glute_bridge.png` | 臀桥训练 |
| `wall_squat.png` | 靠墙静蹲 |
| `straight_leg_raise.png` | 直腿抬高训练 |
| `quad_set.png` | 股四头肌等长收缩 |
| `calf_stretch.png` | 小腿后侧拉伸 |
| `ankle_pump.png` | 踝泵运动 |
| `shoulder_pendulum.png` | 肩关节钟摆运动 |
| `shoulder_external_rotation.png` | 肩外旋弹力带训练 |
| `mckenzie_press_up.png` | 麦肯基俯卧撑（算法扩展） |

搜索关键词见上级目录 `ACTION_MEDIA_GUIDE.md`。
