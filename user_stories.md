# 用户故事文档

## 项目：康健图谱 — 基于大模型的个性化运动康复处方生成系统

---

## 用户角色定义

### 1. 康复患者（Patient）
- **特征**：需要康复指导的患者群体
- **目标**：获得个性化的运动康复处方，改善身体状况

### 2. 康复医生/治疗师（Therapist）
- **特征**：专业的康复医学工作者
- **目标**：快速为患者生成科学、安全的康复处方

### 3. 系统管理员（Administrator）
- **特征**：系统运维和管理人员
- **目标**：管理系统、维护数据库、确保系统稳定运行

---

## 用户故事

### 用户故事 1: 患者查看个性化康复处方
**作为一个** 康复患者  
**我想要** 基于我的身体状况和病历输入一个个性化的运动康复处方  
**以便于** 我能够按照科学的计划进行康复锻炼，加速恢复

**验收标准：**
- ✓ 系统能够接收患者的基本信息（年龄、性别、身高、体重）
- ✓ 系统能够接收患者的康复需求（受伤部位、康复阶段）
- ✓ 系统能够生成详细的康复处方（包括动作、次数、频率）
- ✓ 康复处方需要清晰易懂，包含图文说明

---

### 用户故事 2: 治疗师为患者制定康复计划
**作为一个** 康复医生  
**我想要** 快速地为患者生成符合医学标准的康复处方  
**以便于** 我能够高效地为更多患者提供服务，同时确保处方的科学性和安全性

**验收标准：**
- ✓ 系统提供医生级别的患者信息输入界面
- ✓ 系统能够调用大模型生成符合医学规范的康复处方
- ✓ 医生可以对生成的处方进行审核和修改
- ✓ 系统能够保存处方历史，便于后续追踪

---

### 用户故事 3: 患者追踪康复进展
**作为一个** 康复患者  
**我想要** 查看我的康复进展和数据统计  
**以便于** 我能够了解自己的恢复情况，保持康复动力

**验收标准：**
- ✓ 系统能够记录患者的康复锻炼记录
- ✓ 系统能够生成可视化的进展报表
- ✓ 系统能够提供阶段性康复评估

---

### 用户故事 4: 医生管理患者信息
**作为一个** 康复医生  
**我想要** 管理我的患者信息和康复历史  
**以便于** 我能够快速查找和回顾患者的康复情况

**验收标准：**
- ✓ 系统能够存储和检索患者的基本信息
- ✓ 系统能够记录患者的康复历史
- ✓ 医生可以查看患者列表和患者详情
- ✓ 系统支持患者信息的搜索和筛选

---

### 用户故事 5: 系统管理员维护知识库
**作为一个** 系统管理员  
**我想要** 管理和更新系统中的康复动作知识库  
**以便于** 系统能够不断扩展康复方案的覆盖面和准确性

**验收标准：**
- ✓ 管理员能够添加新的康复动作
- ✓ 管理员能够修改和删除现有的康复动作
- ✓ 管理员能够管理提示词模板
- ✓ 系统能够验证知识库的完整性和合理性

---

### 用户故事 6: 系统反馈康复建议
**作为一个** 康复患者或医生  
**我想要** 基于LLM的个性化反馈和建议  
**以便于** 我能够获得更加科学和人性化的康复指导

**验收标准：**
- ✓ 系统能够基于患者情况给出详细的康复建议
- ✓ 系统能够解释为什么推荐这些动作
- ✓ 系统能够提供安全注意事项和禁忌提示
- ✓ 用户可以对建议进行评分和反馈

---

### 用户故事 7: 患者获取康复教育资源
**作为一个** 康复患者  
**我想要** 获取关于康复运动、日常护理的教育资源  
**以便于** 我能够更好地理解康复过程，配合医生的指导

**验收标准：**
- ✓ 系统提供详细的康复动作说明和演示
- ✓ 系统提供康复知识文章和指南
- ✓ 系统提供常见问题解答（FAQ）
- ✓ 资源清晰易懂，包含视频或图解

---

### 用户故事 8: 医生导出和分享康复方案
**作为一个** 康复医生  
**我想要** 能够导出康复方案并以多种格式分享给患者  
**以便于** 患者能够方便地获取和保存处方信息

**验收标准：**
- ✓ 系统支持导出为PDF格式
- ✓ 系统支持导出为Word格式
- ✓ 系统支持打印功能
- ✓ 导出文档格式美观专业

---

### 用户故事 9: 系统提供统计分析
**作为一个** 医疗机构管理者  
**我想要** 查看系统的使用统计和康复效果分析  
**以便于** 我能够评估系统的价值和改进方向

**验收标准：**
- ✓ 系统能够统计患者数量和康复案例数
- ✓ 系统能够分析各类康复类型的效果
- ✓ 系统能够生成定期报告和数据可视化
- ✓ 支持数据导出功能

---

### 用户故事 10: 用户账户管理
**作为一个** 系统用户  
**我想要** 能够注册、登录和管理我的账户  
**以便于** 我的数据能够被安全保护，个人隐私得到尊重

**验收标准：**
- ✓ 系统支持用户注册和身份验证
- ✓ 用户可以修改密码和个人信息
- ✓ 系统支持忘记密码恢复功能
- ✓ 用户数据加密存储和传输
